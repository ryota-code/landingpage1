# Retro Pixel Animation

レトロなドット絵風アニメーションページです。猫とネズミの追いかけっこが繰り広げられます。

## 機能

- **時間帯切り替え** - 夜明け・あさ・ひる・ゆう・よる の5つの時間帯で背景が変化
- **メッセージ入力** - テキストを入力するとキャンバス上に表示
- **キャラクターアニメーション** - 猫とネズミが出会い、追いかけ、逃げるループアニメーション
- **ピクセルアート風描画** - Canvas APIによるレトロなドット絵表現

## ファイル構成

```
├── index.html   - メインHTML
├── style.css    - スタイルシート
├── script.js    - アニメーションロジック
└── README.md
```

## 使い方

`index.html` をブラウザで開くだけで動作します。
